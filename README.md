# 🎯 OrderLoader 2.0 - Sistema Consolidado

## 📋 **Descripción**

Sistema de automatización SAP consolidado con mejores prácticas. **Versión final limpia y organizada.**

## 🚀 **Instalación y Uso**

### **Instalación**
```bash
cd orderloader
pip install -r requirements.txt
```

### **Uso**
```bash
python main.py
```

### **Testing**
```bash
python test_final.py
```
